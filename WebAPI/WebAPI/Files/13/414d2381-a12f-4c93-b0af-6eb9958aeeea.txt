<h3>Оператор if</h3><p>Оператор if является условным оператором, выполняющим блок кода, если условие верно.<br></br></br>Общая форма оператора if:</p><p>&nbsp;<span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(175, 0, 219);">if</span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;"> (</span><span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(0, 16, 128);">condition</span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;">)</span></p><div style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; line-height: 19px; white-space: pre;"><div>{</div><div>&nbsp;&nbsp;&nbsp;&nbsp;<span style="color: #008000;">// Execute this code when condition</span><br><span style="color: #008000;"> is true</span></div><div>}</div><div><br></br></br></div></div><p>Условием может быть любое выражение, возвращающее true или false. <br></br></br>Например:</p><p><span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(0, 0, 255);">static </span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;"></span><span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(0, 0, 255);">void </span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;"></span><span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(121, 94, 38);">Main</span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;">(</span><span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(0, 0, 255);">string</span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;">[] </span><span style="font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial; color: rgb(0, 16, 128);">args</span><span style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; white-space: pre; text-align: initial;">)</span></p><div style="color: rgb(0, 0, 0); font-family: Consolas, &quot;Courier New&quot;, monospace; font-size: 14px; line-height: 19px; white-space: pre;"><div>{</div><div>&nbsp;&nbsp;<span>&nbsp;</span><span style="color: #0000ff;">int</span><span>&nbsp;</span><span style="color: #001080;">x</span> =<span>&nbsp;</span><span style="color: #09885a;">8</span>;</div><div>&nbsp;&nbsp;<span>&nbsp;</span><span style="color: #0000ff;">int</span><span>&nbsp;</span><span style="color: #001080;">y</span> =<span>&nbsp;</span><span style="color: #09885a;">3</span>;</div><div></div><div>&nbsp;&nbsp;<span>&nbsp;</span><span style="color: #af00db;">if</span> (<span style="color: #001080;">x</span> &gt;<span>&nbsp;</span><span style="color: #001080;">y</span>)</div><div>&nbsp;&nbsp; { </div><div>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span style="color: #001080;">Console</span>.<span style="color: #795e26;">WriteLine</span>(<span style="color: #a31515;">"x is greater </span><br><span style="color: #a31515;">than y"</span>);</div><div>&nbsp;&nbsp; }</div><div>}</div><div><br></br></br></div></div><p>Вышеуказанный код вычислит условие x &gt; y. Если оно верно, будет выполнен код внутри блока if.</p>
